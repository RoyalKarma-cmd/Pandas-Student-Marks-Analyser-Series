# Student Marks Analyzer using Pandas Series

## Project Overview

This project analyzes student marks using **Pandas Series**.

It demonstrates how to perform statistical analysis, identify student performance categories, generate grades, and summarize results using Pandas.

---

## Features

* Store student marks using Pandas Series
* Display original marks
* Statistical analysis
* Find distinguished students
* Find average performers
* Pass/Fail report
* Grade generation
* Project summary
